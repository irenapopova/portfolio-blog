FRONT-END DEVELOPMENT

I focus on valid, semantic HTML that’s search engine optimized and easily maintainable. Stylesheets are strictly organized, and I maintain the current best practices, with modernized CSS using Grid, Flexbox, and Animations. Everything is beautifully responsive and works across a variety of devices and browsers. Frameworks and code-trend-of-the-moment are avoided to ensure all code is easy to maintain and lasts longer. It’s my job to ensure you never have to deal with ugly old code ever again. Organized and semantic HTML and CSS bring your designs to life. I prefer coding from scratch instead of a framework, with no obfuscations to ensure there’s no junk code or bugs. I can write Javascript & jQuery functions from scratch, as well as integrate popular jQuery plugins. I can code static HTML websites as well.
